# Sistema de Mantenimiento Predictivo con Inteligencia Artificial

Proyecto académico desarrollado en la Universidad ECCI para el diseño de un sistema capaz de predecir fallas en maquinaria industrial mediante técnicas de Inteligencia Artificial y Machine Learning.

## Autores
- Jeronimo Espitia Jojoa
- Edwin Enrique Albornoz Archila
- Dilan Felipe Rodríguez Rayo

## Descripción

El sistema analiza datos provenientes de sensores instalados en maquinaria industrial (temperatura, vibraciones, presión, ruido, velocidad y consumo energético) con el objetivo de detectar patrones anómalos que puedan indicar fallas futuras.

Esto permite implementar mantenimiento predictivo, reduciendo costos operativos y evitando paradas inesperadas en la producción.

## Tecnologías

- Python
- Scikit-learn
- TensorFlow
- Pandas
- NumPy
- Flask / FastAPI
- React.js
- Docker

## Estructura del proyecto

- `docs/` → Documentación del sistema
- `src/` → Código fuente
- `data/` → Carpeta para datasets (no incluidos en el repositorio)
- `notebooks/` → Análisis exploratorio

## Objetivo

Diseñar un sistema inteligente que analice datos operativos de maquinaria industrial para anticipar fallas y optimizar el mantenimiento.
