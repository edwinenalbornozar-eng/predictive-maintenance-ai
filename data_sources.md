# Fuentes de Datos

Este proyecto utiliza datasets de mantenimiento predictivo provenientes de fuentes públicas.

Los datasets NO se incluyen en este repositorio debido a restricciones de tamaño y licencias.

## Posibles fuentes de datos

### 1. Kaggle

Predictive Maintenance Dataset  
https://www.kaggle.com/datasets/shivamb/machine-predictive-maintenance

Contiene variables como:

- Temperatura
- Vibración
- Velocidad de rotación
- Torque
- Fallas registradas

### 2. UCI Machine Learning Repository

AI4I 2020 Predictive Maintenance Dataset  
https://archive.ics.uci.edu/ml/datasets/AI4I+2020+Predictive+Maintenance+Dataset

Dataset ampliamente utilizado para investigación académica.

## Uso de datos

Los datos serán utilizados únicamente con fines académicos para el entrenamiento y evaluación de modelos de Machine Learning.
