# Metodología

## 1. Recolección de datos
Sensores registran:

- Temperatura
- Vibración
- Presión
- Ruido
- Velocidad
- Consumo energético

## 2. Procesamiento

- Limpieza de datos
- Normalización
- Almacenamiento

## 3. Entrenamiento del modelo

Algoritmos candidatos:

- Random Forest
- Redes Neuronales
- Isolation Forest
- LSTM

## 4. Predicción

El sistema monitorea continuamente los datos y genera alertas cuando detecta comportamientos anómalos.
