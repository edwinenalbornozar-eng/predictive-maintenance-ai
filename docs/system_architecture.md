# Arquitectura del Sistema

El sistema se compone de cuatro capas principales:

1. Adquisición de datos
Sensores IoT capturan información de la maquinaria.

2. Procesamiento y almacenamiento
Los datos se limpian, normalizan y almacenan en bases de datos.

3. Inteligencia Artificial
Modelos de Machine Learning analizan los datos para detectar anomalías.

4. Visualización
Panel de control que muestra métricas y alertas en tiempo real.
