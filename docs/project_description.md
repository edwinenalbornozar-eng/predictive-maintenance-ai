# Descripción del Proyecto

El proyecto consiste en el desarrollo de un sistema de mantenimiento predictivo basado en inteligencia artificial.

El sistema recopila datos de sensores instalados en maquinaria industrial para analizar patrones de funcionamiento y predecir posibles fallas.

## Problema

Las fallas inesperadas en maquinaria industrial generan:

- Paradas no planificadas
- Altos costos de reparación
- Riesgos de seguridad
- Reducción de productividad

## Solución

Implementar un sistema que utilice algoritmos de Machine Learning para anticipar fallas antes de que ocurran.
